# Goal: Add a frontend to the AI Events Tracker

The goal is to create a functional frontend that displays events from the `events` table in the SQLite database.

## Requirements:
- [x] Implement a backend API (FastAPI) to serve event data.
- [x] Implement a frontend to display events.
- [x] Default view: Only future events.
- [x] Tab/Toggle: View past events.
- [x] Ordering and Filtering capabilities.
- [x] Ensure the frontend is functional and runs.

## Success Criteria:
- The frontend successfully connects to the database via the API.
- Events are correctly sorted and filtered by date.
- Users can toggle between future and past events.
- The UI is clean and usable.
